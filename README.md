# `Программа для прогнозирования аварийных участков на городской улично-дорожной сети`

## *1. Описание*

С помощью данной системы можно анализировать картографические фрагменты карт OSM на предмет аварийности перекрестков, сначала необходимо загрузить .osm файл, затем его выбрать, задать число прогонов и запустить расчет.
Затем сформированная карта может быть экспортирована и детально рассмотрена в этом же окне с индексом аварийности риска.

ПРИМЕЧАНИЕ: для работы системы должно быть предустановлено SUMO GUI на исполняемом устройстве.

## *2. Использованные технологии и среды разработки*

В качестве основного языка программирования выбран Python 3, ниже представлен перечень инструментов и модулей:

+ Микросимуляция и подготовка дорожной сети: SUMO (netconvert, sumo, randomTrips.py), OpenStreetMap; 
+ Графический интерфейс и визуализация: PyQt5, matplotlib; 
+ Обработка данных и параллельные вычисления: NumPy, модули стандартной библиотеки Python (subprocess, xml.etree, concurrent.futures, tempfile, gzip); 
+ Машинное обучение и инференс: PyTorch; 
+ Среда обучения модели: Google Colaboratory с использованием ресурсов GPU.
